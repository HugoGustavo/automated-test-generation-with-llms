# Automated Test Generation with LLMs

This repository contains a comprehensive collection of research articles addressing 11 key research questions (RQ1-RQ12) in the field of Automated Test Generation with Large Language Models (LLMs).

## üìÅ Repository Structure

This repository is organized into CSV files, each addressing a specific research question about LLM-based software testing:

### Research Questions & Corresponding Files

| File | Research Question |
|------|-------------------|
| **`RQ1.csv`** | How do LLM-based tools compare to traditional automated test generation methods in terms of accuracy, coverage, and usability across different testing contexts and application domains? |
| **`RQ2.csv`** | What techniques can be used to validate the accuracy and relevance of LLM-generated tests in diverse software ecosystems? |
| **`RQ3.csv`** | How can LLMs enhance test case design and execution for challenging scenarios like exploratory testing or high-complexity codebases? |
| **`RQ4.csv`** | What strategies improve the coherence and logical flow of test cases generated by LLMs? |
| **`RQ5.csv`** | What are the effects of different prompt engineering strategies on the complexity and depth of test cases produced by LLMs? |
| **`RQ6.csv`** | How does fine-tuning LLMs with domain-specific data influence their ability to generate relevant and accurate test cases? |
| **`RQ7.csv`** | What approaches can standardize the evaluation of LLM-based test generation across different software testing benchmarks? |
| **`RQ8.csv`** | What qualitative methods can complement quantitative metrics in evaluating LLM-based software testing tools? |
| **`RQ9.csv`** | What are the primary limitations and challenges when integrating LLMs into existing software testing pipelines, and how can these be mitigated? |
| **`RQ10.csv`** | How do infrastructure constraints (hardware, cost) affect the scalability of LLM-driven testing in industrial settings? |
| **`RQ11.csv`** | What are the best practices for integrating domain knowledge into LLM-powered testing frameworks, and how does it impact performance? |
| **`RQ12.csv`** | What are the trade-offs between implicit and explicit incorporation of domain knowledge in LLMs for software testing? |

### File Location
All CSV files are located in the `tables/` directory for easy access and organization.
